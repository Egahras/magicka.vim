# magicka.vim
The `magicka` colorscheme for Vim

## Installation (Windows)
   * Stick the `magicka.vim` file in your `~/vimfiles/colors` folder

## Configuration

Just edit your `~/_vimrc` to add:

    syntax on
    colorscheme magicka
